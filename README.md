# BankTalabat - نظام طلبات البنك

هذا التطبيق هو نظام متكامل لتدوير مهام إحضار الطعام وإدارة الطلبات لموظفي البنك، مبني باستخدام Next.js 15 و Firebase و Genkit.

## متطلبات التشغيل المحلي

لشغيل التطبيق على جهاز الكمبيوتر الخاص بك، اتبع الخطوات التالية:

### 1. تثبيت الاعتمادات
قم بتشغيل الأمر التالي في مجلد المشروع:
```bash
npm install
```

### 2. إعداد متغيرات البيئة
قم بإنشاء ملف باسم `.env` في المجلد الرئيسي وأضف المفاتيح التالية:
```env
GEMINI_API_KEY=your_api_key_here
```
*(يمكنك الحصول على المفتاح من Google AI Studio)*

### 3. إعداد Firebase
التطبيق مهيأ حالياً للعمل مع مشروع Firebase الافتراضي. إذا كنت ترغب في استخدام مشروعك الخاص:
1. أنشئ مشروعاً جديداً في [Firebase Console](https://console.firebase.google.com/).
2. فعل خدمة **Firestore Database** و **Authentication**.
3. انسخ إعدادات تطبيق الويب (Web App Config) وضعها في ملف `src/firebase/config.ts`.
4. انسخ قواعد الحماية من ملف `firestore.rules` إلى تبويب Rules في Firestore Console.

### 4. تشغيل التطبيق
للتشغيل في وضع التطوير:
```bash
npm run dev
```

افتح المتصفح على [http://localhost:3000](http://localhost:3000) لمشاهدة التطبيق.

## المميزات الرئيسية
- **نظام تدوير ذكي:** توزيع مهام النزول بين الموظفين بشكل عادل.
- **إدارة طلبات متكاملة:** واجهة سهلة للموظفين لطلب الوجبات.
- **لوحة تحكم للمشرفين:** لإدارة الموظفين، الأقسام، وقائمة الطعام.
- **ذكاء اصطناعي (GenAI):** تحليل الطلبات وتقديم ملخصات ذكية للقسم.
- **تصميم Banking-Style:** واجهة عصرية تشبه التطبيقات البنكية (Neo-Banking).
